Lab 5: Basic PHP

Lab 5: Basic PHP - modify Music Library page into a dynamic PHP page

The Music Library page (music.php) is written to display Scott's MP3s. This page has already been written with the basic HTML and CSS. But the HTML is redundant, and manual page editing is required every time songs are added to the collection.

In this lab, you will turn this page into a dynamic PHP page that removes redundancy and dynamically lists the MP3s on the computer.

Lad 5 exercises

Download the music.zip and complete the following exercises.

  1. Variable for Song Count
  2. Loop for Links
  3. Query Parameter
  4. Array of Favorite Artists
  5. File of Favorite Artists

Ex : Variable for Song Count

Scott has 1234 songs (123 hours of music).
Turn this into a PHP variable you can change.

  • Add a variable for the number of songs, and use it in the HTML/PHP code.
    Change its value to something other than 1234, such as 5678.
    • name the variable as song_count
  • Express the number of hours of music in terms of your variable.
    Assume that 10 songs can be played per hour.
    • hour of music should always be int type.

Ex : Loop for Links

  • The five Billboard news links at the top of the page are redundant, so express them using a for loop.
  • You should be able to change your loop bound to get more/fewer page links.
    • name the loop bound variable as news_pages
  • After the modification, the links should still work when you click on them.

Ex : Query Parameter

  • Add a GET query parameter named newspages for the number of Billboard news links to display.
  • If the parameter is not passed, the page defaults to 5 pages.
music.php?newspages=2 music.php?newspages=4

Ex : Array of Favorite Artists

  • The page lists Scott's favorite artists as a bulleted list.
  • Create an array containing the favorite artists' names and use the array to generate the bullets in the list.
  • Test that it's working by adding at least one more favorite artist (e.g. Queen).

Ex : File of Favorite Artists

  • Instead of writing your own array of favorite artists, read them from a file, favorite.txt.
  • Download this file to your machine and read the artists from it, one per line.
  • Make each artist's name a link.
    • The links have names similar to the artist's name but with underscore.
    • For example, Green Day links to :
      http://en.wikipedia.org/wiki/Green_Day

Ex : Playlists

(See example screenshot on next slide.)

Add code to display the contents of .m3u playlists. A playlist is just a text file containing a list of song file names, one per line. It can also have comment lines that begin with # . (You can ignore/skip comment lines.) For example: 

#EXTM3U
#EXTINF:183,Green Day - Basket Case
basket-case.mp3
#EXTINF:168,Blink 182 - All the Small Things
all-the-small-things.mp3
  • Write code that searches for all playlist .m3u files in the lab5/musicPHP/songs/ directory, and for each one, displays a list of the playlist's songs as a nested list.
    • use a foreach loop and glob call to search for the M3U files
    • use another foreach loop (nested inside) to read each line of a M3U file
    • use strpos function to distinguish comments in M3U file

Ex : Ordering/Sorting

  • Reverse order: Display the playlist in reverse ABC order. You may want to look at PHP's array functions related to sorting.
  • Shuffle: Display the songs for each playlist in a random order each time the page loads.
  • Sort by Size: Display the songs sorted from largest to smallest size rather than in their default sorted order.
    • The largest songs (such as paradise-city.mp3) would go first and smallest songs (such as lucky.mp3) last.
