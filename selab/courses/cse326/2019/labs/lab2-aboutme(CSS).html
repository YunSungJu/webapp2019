    Ex : CSS Styles - Fonts

    Change colors of your AboutMepage:

    • set body font style - text in the body should be displayed in Helvetica, Verdana, or sans-serif font (in this order of precedence)
    • change the font size of the top level heading to 40 point
    • reduce the font size of the text in definition lists to 70%
    • make the definition terms in a definition list bold
    • make the first character of the first word in a paragraph bold and change its font size to 150%

    Ex : CSS Styles - Colors

    Change colors of your AboutMepage:

    • set the color of the page to #939396 and background-color of the page to #1D2475
    • set the color of a link to #FF4F4F
    • set the color of the first and the second level heading to #F7931E
    • set the color of strongly emphasized texts and the definition terms in a definition list to #FFF86B
    • set the color of the quoted text (inline) to #D1DFEB

    Ex : CSS Styles - Text

    Change text style of your AboutMepage:

    • align the texts in lists (both ordered and unordered) and blockquotes to the left
    • remove the text decorations for links (for visited links as well)
    • put underline to a link text when mouse is over it
    • put green text shadow for the first level heading (3px to the right and top)

    Ex : CSS Styles - Table

    Change table style of your AboutMepage:

    • draw a dotted border line for table and every cell in the table
    • merge the border line for table and cells

    Ex : CSS Styles - Advanced

    Create advanced styling of your AboutMepage:

    • change the bullet style for unordered lists to square
    • include a medium thick ridge outline with #F7931E color to images of my favorite things in the page
    • validation link images should not have outline (override style for validation link images)
    • download and add background images to your page - the background image should:
      • be at the top right hand corner of your page
      • not repeat
      • be fixed and not scroll with the rest of the page (background image should always stay at the same position of the page when scrolled)

    Exercise : Validate your style

    Validate the syntax of your CSS code:

    1. open http://jigsaw.w3.org/css-validator/
    2. validate your page by either
      • Validate by File Upload and then browse to your aboutme.css file
      • Validate by Direct Input and then copy.paste your code into the text box
    3. If there are any errors, fix them all
      • try to get the green bar for 0 errors. Warnings are ok
      • if you do not understand the error messages, ask TA for help

    If you finish them all...

    If you finish all the exercises:

    1. you can add any other style you like to your page
    2. check the correctness of your page with TA if necessary
    3. submit your css file to your Git repository under lab02 folder
